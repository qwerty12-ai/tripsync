- ⚡ Real-time UI Updates  
- ✏️ Full CRUD Operations  

---

## 🧠 Why TripSync?

Most travel planning tools are scattered and unstructured.  
TripSync introduces a **hierarchical, structured approach**:
        User → Trip → Destination → Activity  

This ensures:
- Better clarity  
- Easier management  
- Seamless collaboration

## 🧱 Tech Stack

### Frontend
- React (Hooks)
- Tailwind CSS
- Axios  

### Backend
- Spring Boot
- REST APIs
- JPA (Hibernate)  

### Database
- MySQL  

---

## 🏗️ System Architecture

      Frontend (React) 
            ↓ 
    Spring Boot REST API 
            ↓ 
      MySQL Database 

---

## ⚙️ Run Locally

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend 
```bash
cd frontend
npm install
npm run dev
```
